# IrisCinema API

## Technologies Used

- ASP.NET Core Web API
- Entity Framework Core
- AutoMapper
- Middleware for control exception
- SQLite

## Features

- Create rooms
- Create sessions
- Create reservations  
- Cancel reservations  
- Seat management
- Prevent double booking

## Getting Started

### Clone the repository

git clone https://github.com/rmassaretto/IrisCinema-Prova.git

### Restore dependencies

dotnet restore

### Restore database

database-update

### Run the application

dotnet run
